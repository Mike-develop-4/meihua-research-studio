# 观象 · 梅花易数研究台

一个可解释、可复核的梅花易数起卦与解卦研究工具。项目重点不是简单输出“吉凶”，而是把本卦、互卦、变卦、体用生克、月令旺衰、体党用党、动爻转折和问事语境组织成可追溯的证据链。

## 主要功能

- 六种起卦方式：时间、报数、文字、物数、物象方位、自定义排盘。
- 三步向导：明确问事语境、选择取数方式、复核数源与卦象。
- 展示本卦、互卦、变卦及完整计算过程。
- 固定本卦体卦，分别分析用卦、上下互和变卦对主体的作用。
- 结合月令旺相休囚死修正生克关系的严重程度。
- 提供综合判断、矛盾化解、证据权重、行动建议和应期参考。
- 支持项目、事业、财务、关系、学业、寻物、健康和一般问事。
- 本地保存卦例、收藏、两卦对照、复制报告和打印。
- 响应式桌面与移动端界面。

## 解释原则

- 本卦体卦由动静确定后保持固定。
- 互卦上下互分别与固定体卦比较，不在互卦内部重新定义主体。
- “用克体”必须结合体用双方旺衰判断，不能直接等同于失败。
- 本卦、互卦、变卦是连续证据，不机械等同于确定日期上的开始、过程和结局。
- 趋势分值只用于整理传统象数信息，不代表客观概率。
